# 安装 Mod

## 安装

大部分 Mod 的安装非常简单，只需要两步

1. 从 NexusMods 下载 Mod
2. 使用 MO2 安装

## 示例 (未关联NexusMods / N 网之外的 Mod)

直接将下载后的 Mod 压缩文件拖入 MO2 即可。

<VideoElement relativePath="install-mods-2.webm" />

## 示例 (已关联NexusMods)
> 此方法仅推荐在安装小型 Mod 时使用，在下载大型 Mod 时 MO2 可能因文件过大或网络不稳定而出现下载问题。

将 Mod 从右侧下载面板拖入左侧即可。

关于 MO2 如何关联 NexusMods，参考 [MO2 参考](/docs/references/mod-organizer/#关联-nexusmods)
