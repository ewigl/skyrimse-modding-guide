# 安装 Mod

大多数 Mod 只需要简单两步即可安装：

1. 从 [NexusMods](https://www.nexusmods.com) 等网站下载所需 Mod 文件。
2. 使用 Mod Organizer 2（MO2）安装。

## 📁 手动下载

- 下载 Mod 压缩包（通常为 `.zip`、`.rar` 或 `.7z` 格式）
- 将压缩文件直接拖入 MO2 界面，MO2 会自动识别并添加到列表中

## 🌐 通过 MO2 下载

> ⚠️ **建议仅用于安装体积较小的 Mod**，
> MO2 在下载大型 Mod 时可能因网络不稳定或服务端限制而发生失败或中断。

前提：MO2 已正确关联 NexusMods API 授权。
📎 参考文档：[MO2 参考 / 关联 NexusMods](/docs/references/mod-organizer/#关联-nexusmods)

操作方式：

- 在 NexusMods 页面点击“Mod Manager Download”下载
- 在 MO2 中，从右侧“下载”面板拖动对应 Mod 至左侧“Mod 列表”面板，完成安装
