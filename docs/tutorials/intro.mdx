# 前奏

<span className='indented_paragraph'>
对于天际 Mod 新手来说，很容易会因不断出现的"玄学"问题而不知所措。谷歌等搜索引擎也很可能找不到想要的答案。

但是，其实只要对 Mod 的结构和功能有基本的了解，就可以轻松地解决大多数问题或从一开始就避免出现问题。所需要的只是一些时间、耐心和一些阅读。

这就是问题所在。学习如何 Mod 需要时间和精力。野心越大，Mod 列表就越大、越复杂，需要的时间和精力就越多。Mod 与其他任何事情一样，需要一定程度的投入。

**本教程仅支持 Steam 最新游戏版本 `1.6.1170`，但是仍然可以作为其他版本游戏的参考教程。**
</span>
