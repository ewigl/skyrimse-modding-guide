# 前奏

欢迎来到 SMGL 教程！

本教程将带你从零开始，一步步为《上古卷轴 5：天际》安装 Mod。
教程覆盖从准备工具到实战操作的完整流程，不需要专业知识，除了计算机基础常识以及基本的**阅读理解以及思考能力**。

本教程基于 **Steam 最新游戏版本 `1.6.1170`** 编写，其他游戏版本（如 GOG、Steam 旧版）也可参考使用（毕竟原理相同），部分步骤可能略有差异。

点击下一页，将开始进行必要的准备工作，为后续 Mod 安装打下基础。

:::tip
建议边阅读边操作，避免遗漏任何步骤。
:::
